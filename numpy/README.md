# NumPy 100 Project

This folder contains filled  **NumPy 100 exercise**, created to demonstrateproficiency with **NumPy**, a core Python library for numerical computing. The project includes 100 exercises that cover a wide range of NumPy functionalities.

## Project Overview

- **100 exercises** covering:
  - Array creation and manipulation
  - Mathematical operations
  - Indexing, slicing, and broadcasting
  - Statistical computations
  - Linear algebra operations
  - Random number generation
- Designed to show **efficient and Pythonic usage** of NumPy.
